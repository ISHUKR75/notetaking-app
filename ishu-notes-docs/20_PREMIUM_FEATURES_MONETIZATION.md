# Ishu Notes — Premium Features & Monetization

## Pricing Philosophy

**"Free should be genuinely useful. Premium should be genuinely worth it."**

The free tier is not crippled — it's a complete note-taking experience. Premium adds power-user features, cloud sync, collaboration, and AI that transform Ishu Notes from great to exceptional.

- **Fountain Pen** — Velocity-sensitive ink flow, wet/dry effect

- **Calligraphy Pen** — Flat nib, angle-dependent width
