# Ishu Notes — Complete Keyboard Shortcuts Reference

## Overview

Ishu Notes has **100+ keyboard shortcuts** across all platforms. Every shortcut is customizable from Settings → Keyboard Shortcuts. Shortcuts follow platform conventions: Ctrl on Windows/Linux, Cmd on macOS.

**Notation:** `Mod` = Cmd on macOS, Ctrl on Windows/Linux

## Shortcut Customization

```
Settings → Keyboard Shortcuts:

┌─────────────────────────────────────────────────────────────┐
│  🔍  Search shortcuts...                                    │
├──────────────────────────────────────┬──────────────────────┤
│  ACTION                              │  SHORTCUT            │
├──────────────────────────────────────┼──────────────────────┤
│  New Note                            │  ⌘N         [Edit]  │
│  Open Search                         │  ⌘F         [Edit]  │
│  Toggle Dark Mode                    │  ⌘⇧L        [Edit]  │
│  Bold                                │  ⌘B         [Edit]  │
│  Open AI Assistant                   │  ⌘⇧A        [Edit]  │
│  ...                                 │                      │
├──────────────────────────────────────┴──────────────────────┤
│  [Reset all to defaults]                                    │
└─────────────────────────────────────────────────────────────┘
```

Rules:
- Any shortcut can be reassigned
- Conflicts are highlighted in red
- Character shortcuts (like `P` for pen) can be disabled globally
- Export/import custom shortcut profiles
